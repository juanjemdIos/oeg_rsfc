About RSFC

RSFC (Research Software Fairness Checks) is a command‑line tool developed at the Ontology Engineering Group – Universidad Politécnica de Madrid.

Given a GitHub or GitLab repository URL, RSFC performs a series of checks based on a list of research software quality indicators (RSQI).

Included Quality Indicators (RSQI)

  • Software has license
  • Software has citation
  • Has releases
  • Repository workflows
  • Version control usage
  • Requirements specified
  • Software documentation
  • Persistent & unique identifier
  • Descriptive metadata
