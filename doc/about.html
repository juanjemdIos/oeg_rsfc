About RSFC

RSFC (Research Software Fairness Checks) is a command‑line tool a Python library developed at the Ontology Engineering Group – Universidad Politécnica de Madrid.

Given a GitHub or GitLab repository URL, RSFC performs a series of checks based on a list of research software quality indicators (RSQI).

Included Quality Indicators (RSQI)

  • Software has license
  • Software has citation
  • Has releases
  • Repository workflows
  • Version control usage
  • Requirements specified
  • Software documentation
  • Persistent & unique identifier
  • Descriptive metadata
  • Versioning standards use
  • Software is archived in Software Heritage
  • Software provides tests

Covered FAIR Software Metrics (FRSM)

  • FRSM‑1: Does the software have a globally unique and persistent identifier?
  • FRSM‑3: Does each version of the software have a unique identifier?
  • FRSM‑4: Does the software include descriptive metadata which helps define its purpose?
  • FRSM‑5: Does the software include development metadata which helps define its status?
  • FRSM‑6: Does the software include metadata about the contributors and their roles?
  • FRSM‑7: Does the software metadata record include the identifier for the software?
  • FRSM‑8: Does the software have a publicly available, openly accessible and persistent metadata record?
  • FRSM‑9: Is the software developed in a code repository / forge that uses standard communications protocols?
  • FRSM‑13: Does the software describe what is required to use it?
  • FRSM‑14: Does the software come with test cases to demonstrate it is working?
  • FRSM‑15: Does the software source code include licensing information for the software and any bundled external software?
  • FRSM‑16: Does the software metadata record include licensing information?
  • FRSM‑17: Does the software include provenance information that describe the development of the software?
