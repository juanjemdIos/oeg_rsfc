About RSFC

RSFC (Research Software Fairness Checks) is a command‑line tool a Python library developed at the Ontology Engineering Group – Universidad Politécnica de Madrid.

Given a GitHub or GitLab repository URL, RSFC performs a series of checks based on a list of research software quality indicators (RSQI).
